# 📅 Rezervasyon Sistemi (PHP MySQL)

## 📌 Proje Hakkında
Bu proje, kullanıcıların sisteme kayıt olup giriş yaptıktan sonra rezervasyon oluşturabildiği basit bir web uygulamasıdır.

Kullanıcılar tarih ve saat seçerek rezervasyon ekleyebilir, mevcut rezervasyonlarını görüntüleyebilir ve silebilir.

---

## 🚀 Özellikler

- 👤 Kullanıcı kayıt sistemi
- 🔐 Giriş / çıkış işlemleri (session yapısı)
- 📅 Rezervasyon ekleme
- 📋 Rezervasyon listeleme
- 🗑️ Rezervasyon silme
- 🎨 Basit ve modern arayüz (Bootstrap)

---

## 🛠️ Kullanılan Teknolojiler

- PHP (OOP + Procedural)
- MySQL
- HTML
- CSS
- Bootstrap 5
- JavaScript (basit kullanım)
